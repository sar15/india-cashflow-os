# Import Templates

Use these files as the first manual onboarding path when Tally or Zoho data is not ready.

- `setup_template.csv`: opening cash and as-of date
- `cash_events_template.csv`: receivables, payables, and manual cash events
- `obligations_template.csv`: recurring obligations such as GST, payroll, TDS, EPF, EMI, and rent

Recommended flow:

1. Fill `setup_template.csv`
2. Fill `cash_events_template.csv`
3. Fill `obligations_template.csv`
4. Convert them into an `.xlsx` workbook with sheets named `Setup`, `Cash Events`, and `Obligations`
